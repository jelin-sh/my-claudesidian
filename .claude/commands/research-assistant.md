# 研究助手

通过搜索 vault 并综合发现来对主题进行深入研究。

## 流程

1. **初步搜索**
   - 在整个 vault 中搜索该主题
   - 识别所有相关笔记
   - 记录现有知识中的空白

2. **深入探索**
   - 彻底阅读所有相关笔记
   - 提取关键洞察和引用
   - 识别矛盾或张力
   - 绘制想法之间的连接

3. **综合**
   - 创建发现摘要
   - 突出模式和主题
   - 记录仍未回答的问题
   - 建议进一步研究的领域

## 输出结构

```markdown
# 研究摘要：[主题]

## 现有知识

- vault 中已有的内容
- 以前工作的关键洞察

## 关键主题

1. 主题 1
   - 支持笔记：[[笔记1]]、[[笔记2]]
   - 关键洞察：...
2. 主题 2
   - 支持笔记：[[笔记3]]、[[笔记4]]
   - 关键洞察：...

## 矛盾/张力

- 想法冲突的地方
- 未解决的问题

## 空白

- 缺少什么
- 下一步研究什么

## 连接

- 相关主题：[[主题1]]、[[主题2]]
- 令人惊讶的链接：...

## 建议的下一步

1. 需要的具体研究
2. 要探索的问题
3. 要尝试的实验
```

## 提示

- 最初要广泛撒网，然后再聚焦
- 寻找意想不到的连接
- 不要忽视矛盾 - 洞见往往存在于矛盾之处
- 始终建议具体的下一步行动
