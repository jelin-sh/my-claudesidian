# 每日回顾

在一天结束时进行回顾，记录进展并为明天做好准备。

## 回顾流程

1. **今日活动**
   - 查找今天修改的所有笔记
   - 识别创建的新笔记
   - 审查所有项目的工作

2. **进展评估**
   - 完成了什么？
   - 遇到了什么阻碍或阻塞？
   - 出现了什么意外的发现？

3. **捕捉洞察**
   - 今天的关键学习
   - 发现的新连接
   - 提出的问题

4. **明天准备**
   - 前 3 项优先事项
   - 需要完成的未闭环
   - 需要探索的问题

## 输出格式

创建或更新每日笔记：

```markdown
# 每日回顾 - [日期]

## 已完成

- ✓ [已完成项目 1]
- ✓ [已完成项目 2]

## 取得进展

- [项目/领域]：[推进内容]
- [项目/领域]：[推进内容]

## 洞察

- [关键认识或连接]
- [重要学习]

## 受阻/停滞

- [未取得进展的内容及原因]

## 发现的问题

- [出现的新问题]
- [需要研究的内容]

## 明日重点

1. [优先事项 1]
2. [优先事项 2]
3. [优先事项 3]

## 未闭环项

- [ ] [需要记住的事项]
- [ ] [需要跟进的人]
- [ ] [需要发展的想法]
```

## 后续行动

- 将已完成的项目任务归档
- 更新项目状态笔记
- 链接相关的发现
- 标记需要注意的项目
