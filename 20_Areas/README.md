# 🔄 领域（Areas）

你随时间推移的持续责任和活动范围。

## 用途

领域是**持续的责任**，它们：
- 没有结束日期
- 需要持续关注
- 定义你的角色和承诺
- 需要定期维护

## 什么属于这里

### 个人领域
- 健康与健身
- 财务
- 人际关系
- 个人发展
- 家庭与环境

### 职业领域
- 团队管理
- 客户关系
- 职业技能
- 行业知识
- 社交网络

### 创作领域
- 写作练习
- 内容创作
- 学习与研究
- 副业项目

## 领域结构

每个领域可能包含：
```
领域名称/
├── README.md          # 概述和原则
├── Goals/            # 当前重点领域
├── Resources/        # 参考资料
├── Reviews/          # 定期检查
└── Ideas/           # 未来可能性
```

## Claude Code 工作流

### 领域回顾
```
查看我的 [领域名称] 领域。
你在近期笔记中看到什么模式？
哪些方面需要更多关注？
```

### 设定目标
```
帮助我为 [领域] 设定季度目标。
基于近期活动，我应该优先考虑什么？
```

### 发现空白
```
分析我的 [领域] 笔记。
有哪些重要主题我没有追踪？
我应该问什么问题？
```

### 跨领域分析
```
比较我的 [领域 A] 和 [领域 B] 笔记。
它们在哪里重叠？
它们如何相互启发？
```

## 领域 vs 项目

**问自己**：这有结束日期吗？

| 领域（持续进行） | 项目（有时间限制） |
|---------------|---------------------|
| "健康" | "六月前减重 10 磅" |
| "财务" | "创建 2024 预算" |
| "写作" | "完成博客文章系列" |
| "学习" | "完成 Python 课程" |
| "客户" | "交付项目 X" |

## 维护实践

### 每周
- 快速扫描每个领域
- 记录任何紧急需求
- 捕捉新想法

### 每月
- 审查目标和进度
- 更新资源列表
- 清理旧笔记

### 每季度
- 与 Claude Code 深度回顾
- 调整重点领域
- 归档过时材料

## Claude Code 提示

### 健康检查
```
对我所有领域进行健康检查。
哪些有近期活动？
哪些被忽视了？
```

### 平衡评估
```
看看我的领域，我花费精力最多在哪里？
什么不平衡？
```

### 整合机会
```
在我的不同领域之间找到联系。
更好的整合在哪里可以创造价值？
```

## 提示

- **领域启发项目** - 项目通常从领域需求中产生
- **保持活跃** - 不活跃的领域应该归档
- **定期回顾** - 领域在没有关注时会偏离
- **记录标准** - "好"的样子是什么？
- **追踪指标** - 即使定性领域也能从测量中受益

## 记住

领域是你生活的背景。它们不会完成，但会演进。使用 Claude Code 帮助你看到模式并在所有责任之间保持平衡。
